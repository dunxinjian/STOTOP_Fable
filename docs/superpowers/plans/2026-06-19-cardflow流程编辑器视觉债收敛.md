# CardFlow 流程编辑器视觉债收敛（橙化+去绿）Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 把 CardFlow 流程定义编辑器模块（主页 + 8 designer 内件 + 列表页，共 10 文件）的 ~367 处裸色彩收编进全局令牌——绿青伪主色橙化、绿染中性去绿、真语义色归位——并清理字形/复用缺口，消除「主页改了内件没改」的割裂。

**Architecture:** 纯前端视觉债收敛，零后端、零新增令牌（复用 `--color-primary` 体系 + `color-mix` 表达低透明梯度）。按 spec §5 映射表做确定性替换；每文件验证 = `rg` 色彩复扫归零 + `stylelint` 0 报错 + `vite build` 绿。结构性重做（拆文件/外壳迁移）与状态快筛**不在本计划**（另排期）。

**Tech Stack:** Vue 3 + Ant Design Vue + SCSS；设计令牌真源 `web/src/stores/theme.ts` / `web/src/styles/variables.scss`；共享组件 `PageHeader`/`BaseCard`/`StatusTag`/`EmptyState`。

**上游契约（只读，照做）：** spec `docs/superpowers/specs/2026-06-19-cardflow流程编辑器视觉债收敛-design.md`（§5 是映射权威）、`web/docs/PATTERNS.md`、`web/docs/TOKENS.md`。

**验证约定（全计划通用）：**

## Task 2: 主页色彩三分映射（197 hex + 36 rgba → 令牌）

**Files:**
- Modify: `web/src/views/cardflow/FlowDefinitionEditPage.vue`（`<style>` 块 + 行 1775 内联）

**映射权威 = spec §5。** 按「橙化 / 去绿中性 / 语义」三分处理每一处色彩。**必须遵守的关键规则：**

1. **橙化（绿青→主色）**：`#1f6f5f`（实色）→ `var(--color-primary)`；低透明 `rgba(31,111,95,.08~.1)`（选中/激活底）→ `var(--color-primary-light)` 或 `color-mix(in srgb, var(--color-primary) 8%, transparent)`（按原 % 取档）；`rgba(31,111,95,.18~.55)`（焦点环/描边/阴影）→ `var(--color-primary-border)` 或 `color-mix` 对应档；**画布网格线** `rgba(31,111,95,.05)` → 中性 `color-mix(in srgb, var(--text-3) 10%, transparent)`（**不染橙**）。
2. **去绿中性**：绿染与纯灰统一——文字 `#1f3029/#22332c/#18241f/#111827/#1f2937`→`var(--text-1)`；`#718078/#66756e/#74817a/#4b5563/#667085`→`var(--text-2)`；`#5f6f67/#a6b0ab/#9ca3af/#8b95a1`→`var(--text-3)`；边 `#d7e3df/#e6ebe8/#bfd2c9/#e5e7eb/#eef0f3`→`var(--border)`（强边 `#c4cad3/#d4d8e0`→`var(--border-strong)`）；底 `#f7faf9/#f8fbfa/#edf1ef/#fafbfc/#f8fafc`→`var(--bg-page)` 或 `var(--bg-muted)`；`#fff/#ffffff`→`var(--bg-card)`；阴影 `rgba(23,37,31,.08)/rgba(0,0,0,.06)/rgba(15,23,42,.04)`→`var(--shadow-sm)`，`rgba(25,39,32,.12)`→`var(--shadow-md)`。
3. **语义色（保留语义，只换源）**：`#16a34a/#1f9d55/#edf7f2`→`var(--color-success)`/`-text`/`-light`；`#2563eb/#1d4ed8/#eff6ff/#bfdbfe/#1e40af`→`var(--color-info)`/`-light`/`-text`；暖金 `#fffaf0/#f0c98e/#8a5e14/#b45309/#d97706`→`var(--color-warning-light)`/`-warning`/`-text`（**这族是 warning 语义，勿当装饰中性化**）；`#ef4444/#fecaca/#fff8f8/rgba(239,68,68,.x)`→`var(--color-danger)`/`-light`/`-border`。
