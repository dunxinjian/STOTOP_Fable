**关键事实（贯穿全计划，先记住）：**
- `ApiResult<T>.Success(data)` 是泛型成功工厂（**不是 `.Ok()`**）；`ApiResult.Fail(msg)` 失败。
- orgId 一律 `private long GetOrgId() => (long)(HttpContext.Items["CurrentOrgId"] ?? 0L);`，service 首参 `long orgId`。
- 四张表无具名 DbSet，一律 `_db.Set<QlShentongXxx>()`；DbContext 类型 `STOTOPDbContext`（`STOTOP.Infrastructure.Data`），注入字段名 `_db`。
- 权限特性 `[RequirePermission(QualityPermissions.Xxx)]`（命名空间 `STOTOP.Module.System.Filters`）。
- 复杂 LINQ（GroupBy + Distinct + Count、跨期合成）先 `ToListAsync()` 拉内存再算，避免 EF 翻译失败——这是本仓库既定惯例。
- 每次 commit 消息用中文，并以仓库约定的 `Co-Authored-By: Claude Opus 4.8 <noreply@anthropic.com>` 结尾。
- 前端 type-check 基线本就红，**验证前端以 `npm run build`（vite build）兜底**，不拿 `vue-tsc` 当门禁。

**已确认口径（来自 design spec §1/§3）：**
- 率值类 KPI（签收率/出仓及时率/滞留率/虚签率/遗失率ppm）= **区间末日快照**；多网点时末日当天按 **件量加权**（权重取 `F日均出港量`）。
- 累计量/金额类（考核金额合计、问题件数、各域预估金额）= **区间累计 SUM**。
- 趋势图 = 日粒度逐日打点（每天率值多网点件量加权），独立于 KPI 卡。
- 签收率 KPI 主显 `F当天及时签收率`、副显 `F48h签收率`。
- 红黑榜"问题件" = 质量事件计数（按 `F员工工号` 聚合事件表）；客诉=`F客诉发起量`、虚签=`F虚假签收数`、超时=`F派送超时T0..T3数`之和、考核金额=`F考核金额合计`。
- 待认领事件 = `F员工匹配状态 ∈ {0,3}`，视图3 灰显+可筛+认领页角标。
- 多域命中 = 同一 `F运单号` 命中 ≥2 个 distinct `F质量域`。

using STOTOP.Core.Models;
using STOTOP.Infrastructure.Data;
using STOTOP.Module.Quality.Dtos.CarrierDashboard;
using STOTOP.Module.Quality.Entities;

namespace STOTOP.Module.Quality.Services.CarrierDashboard;

public class CarrierQualityDashboardService : ICarrierQualityDashboardService
{
    private readonly STOTOPDbContext _db;

    public CarrierQualityDashboardService(STOTOPDbContext db)
    {
        _db = db;
    }

    // ───────────────────────── 纯函数口径（可单测）─────────────────────────

    /// <summary>件量加权平均：Σ(value*weight)/Σweight；权重和为0或全空时退算术平均；无有效值返回 null。</summary>
    public static decimal? WeightedAverage(IEnumerable<(decimal? Value, decimal? Weight)> items)
    {
        decimal sumVW = 0m, sumW = 0m, sumV = 0m;
        int n = 0;
        foreach (var (v, w) in items)
        {
            if (v is null) continue;
            n++;
            sumV += v.Value;
            var weight = w ?? 0m;
            sumVW += v.Value * weight;
            sumW += weight;
        }
        if (n == 0) return null;
        return sumW > 0m ? sumVW / sumW : sumV / n;
    }

    /// <summary>期间累计：忽略 null 求和。</summary>
    public static decimal PeriodSum(IEnumerable<decimal?> values)
    {
        decimal s = 0m;
        foreach (var v in values) if (v.HasValue) s += v.Value;
        return s;
    }

    /// <summary>派送超时合计：T0..T3 求和，null 当 0。</summary>
    public static int TimeoutTotal(int? t0, int? t1, int? t2, int? t3)
        => (t0 ?? 0) + (t1 ?? 0) + (t2 ?? 0) + (t3 ?? 0);

```csharp
    [HttpGet("events/export")]
    [RequirePermission(QualityPermissions.CarrierQualityView)]
    public async Task<IActionResult> ExportEvents([FromQuery] EventQuery query)
    {
        query.Page = 1;
        query.Size = 100000; // 一次性全量（按期上千行，足够）
        var result = await _service.GetEventsAsync(GetOrgId(), query);
        var rows = result.Data?.Items ?? new List<QualityEventRowDto>();

        string Esc(string? s)
        {
            s ??= "";
            return s.Contains(',') || s.Contains('"') || s.Contains('\n') ? $"\"{s.Replace("\"", "\"\"")}\"" : s;
        }

        var sb = new System.Text.StringBuilder();
        sb.Append('﻿'); // BOM，保 Excel 正确识别 UTF-8 中文
        sb.AppendLine("业务日期,运单号,网点编码,网点名称,员工工号,员工姓名,质量域,问题类型,严重度,是否考核,考核金额,平台,待认领,多域命中");
        foreach (var r in rows)
        {
            sb.AppendLine(string.Join(',', new[]
            {
                Esc(r.Date?.ToString("yyyy-MM-dd")), Esc(r.Waybill), Esc(r.NetworkCode), Esc(r.NetworkName),
                Esc(r.EmpNo), Esc(r.EmpNameRaw), Esc(r.Domain), Esc(r.ProblemName), r.Severity.ToString(),
                r.IsAssessed ? "是" : "否", (r.Fee ?? 0m).ToString("0.00"), Esc(r.Platform),
                r.IsPending ? "是" : "否", r.IsMultiDomain ? "是" : "否"
            }));
        }

        var bytes = System.Text.Encoding.UTF8.GetBytes(sb.ToString());
        var fileName = $"问题件追踪-{DateTime.Now:yyyyMMdd-HHmmss}.csv";
        return File(bytes, "text/csv;charset=utf-8", fileName);
    }
```

> ⚠️ 上面 `sb.Append(...)` 那行追加的是一个**不可见的 U+FEFF BOM 字符**（保证 Excel 识别 UTF-8 中文）。从本文档复制时该字符易丢失。**最稳妥的写法**：不要在 StringBuilder 里塞 BOM 字符，改为在编码后用字节前缀拼 BOM——把 `var bytes = System.Text.Encoding.UTF8.GetBytes(sb.ToString());` 替换为下面两行，并删掉那条 `sb.Append(...)` BOM 行：
>
> ```csharp
> var body = System.Text.Encoding.UTF8.GetBytes(sb.ToString());
> var bytes = new byte[] { 0xEF, 0xBB, 0xBF }.Concat(body).ToArray(); // UTF-8 BOM 字节前缀，复制安全
> ```
>
> （需 `using System.Linq;`，该文件顶部已隐式含；若报缺失则补上。）
